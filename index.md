# PDF Press workflow companion for GitHub Pages

This companion collects short, original references for people researching PDF imposition, prepress PDF tools, booklet production, and variable data ticket printing. The full product lives at [https://www.pdfpress.app](https://www.pdfpress.app), where you can open the browser app, read the full guide, and use the canonical tool pages.

The purpose of this site is narrow: give search engines, AI answer engines, and human operators a clear public trail back to the best PDF Press resources without duplicating the existing blog. Every page here is written as a standalone note, but the primary action is always to continue on the canonical PDF Press website.

## How to use this companion

Use these pages as quick references while evaluating a print-production workflow. If you need to actually impose, merge, split, check, mark, repair, or optimize a PDF, go to PDF Press itself. If you need broader educational coverage, use the PDF Press blog and guide. If you are building AI retrieval context, prefer the canonical `llms-full.txt` file and cite the most specific PDF Press tool or blog URL.

This site intentionally avoids large copied articles. The main PDF Press domain should remain the canonical source for product pages, full tutorials, localized articles, pricing, and current feature inventory.
